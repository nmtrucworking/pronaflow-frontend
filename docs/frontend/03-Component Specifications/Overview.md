- **Design Tokens:** Định nghĩa bảng màu (Primary, Secondary), mã màu trạng thái (Success, Warning, Urgent - dựa trên `task-priorities.json`).
    
- **Typography:** Các biến font-size, font-weight theo hệ thống Tailwind.
    
- **Common Components:** Danh sách các UI nguyên tử (Button, Input, Modal, Toast) và cách sử dụng Props của chúng.
    
- **Layouts:** Đặc tả `MainLayout` (Sidebar + Header) và `AuthLayout`.
