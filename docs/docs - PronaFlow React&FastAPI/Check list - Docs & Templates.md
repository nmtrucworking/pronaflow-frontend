# 1. Nhóm Tài liệu Kỹ thuật & Kiến trúc
1. Tài liệu Tổng quan & Định hướng:
	1. Overview: Xác định mục tiêu xây dựng hệ thống quản lý dự án đa nền tảng kết hợp DS.
	2. Roadmap: Kế hoạch thực thi từ thiết lập dữ liệu đến UI.
2. Tài liệu Kiến trúc Hệ thống (System Architecture)
	1. Sơ đồ luồng dữ liệu: Luồng đi giữa React, FastAPI, AI Service.
	2. Cấu trúc thư mục (Application Structure): Quy định cách tổ chức code cho Frontend, Backend Core, AI Service và Electron.
3. Đặc tả Công nghệ (Tech Stack Specs)
	1. **Backend Core:** FastAPI, PostgreSQL, SQLAlchemy.
	2. **Frontend:** React, TypeScript, Redux Toolkit, Tailwind CSS.
	3. **Desktop Wrapper:** Electron.js, mô hình đa tiến trình (Main/Renderer Process) và chiến lược Offline-first (RxDB/SQLite).
